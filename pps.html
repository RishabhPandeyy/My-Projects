Programming for Problem Solving (BCS101/BCS201) Syllabus - AKTU

Unit I: Introduction to Programming

Computer Fundamentals

  • Computer system and its components
  • Computing environments
  • Computer languages
  • Creating and running programs

Problem-Solving Techniques

  • Algorithms
  • Flowcharts

Introduction to C Language

  • History and basic structure of C programs
  • Process of compiling and running a C program
  • C tokens: keywords, identifiers, constants, strings, special symbols
  • Variables and data types
  • Input/output statements

Unit II: Operators, Expressions, and Control Structures

Operators and Expressions

  • Arithmetic, relational, logical, assignment, increment/decrement, bitwise, conditional, and special operators
  • Operator precedence and associativity
  • Evaluation of expressions
  • Type conversions in expressions

Control Structures

  • Decision statements: if, if-else, nested if-else, switch
  • Loop control statements: while, for, do-while
  • Jump statements: break, continue, goto

Unit III: Arrays and Functions

Arrays

  • One-dimensional arrays: declaration, initialization, and operations
  • Two-dimensional arrays: declaration, initialization, and operations
  • Multidimensional arrays

Basic Algorithms

  • Searching algorithms
  • Sorting algorithms: bubble sort, insertion sort, selection sort

Functions

  • User-defined and built-in functions
  • Storage classes
  • Parameter passing: call by value, call by reference
  • Passing arrays to functions
  • Recursion: factorial, Fibonacci series, Ackermann function, quick sort, merge sort

Unit IV: Strings and Pointers

Strings

  • Arrays of characters
  • Variable-length character strings
  • Inputting character strings
  • Character library functions
  • String handling functions

Pointers

  • Pointer basics and arithmetic
  • Pointers to pointers
  • Generic pointers
  • Array of pointers
  • Functions returning pointers
  • Dynamic memory allocation

Unit V: Structures and File Handling

Structures and Unions

  • Structure definition, initialization, and accessing
  • Nested structures
  • Arrays of structures
  • Structures and functions
  • Self-referential structures
  • Unions
  • Typedef and enumerations

File Handling

  • Command-line arguments
  • File modes
  • Basic file operations: read, write, append

Note: This syllabus (BCS101/BCS201) is based on the AKTU curriculum. Refer to official sources for the most current version.
